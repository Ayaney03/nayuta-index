# 配信文章管理

## 配信完了チェック
配信完了したファイルには先頭に「✅」マークを付ける

## ファイル命名規則
- LINE_MMDD_配信内容.txt
- なゆたMMDDLINE.txt / なゆたMMDDメルマガ.txt

## 配信タイミング
### 公式LINE
- 朝: 7:00-8:00
- 昼: 12:00-13:00  
- 夕: 18:00-19:00
- 夜: 21:00-22:00

### なゆたメルマガ
- 基本: 21:00配信
- スペース等特別配信: 21:10配信

## 注意事項
- 配信前に必ず内容確認
- リンク切れチェック
- 誤字脱字チェック
- 配信対象者確認
