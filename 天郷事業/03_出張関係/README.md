# 出張関係資料

## 📁 フォルダ概要

このフォルダは、出張・移動に関連する資料・計画・経費管理を行います。

## 📋 管理対象

### 移動計画
- 出張スケジュール
- 交通手段の予約・管理
- ルート計画
- 時間調整

### 宿泊関係
- ホテル・宿泊施設の予約
- 宿泊費管理
- アメニティ・設備確認

### 経費管理
- 交通費計算
- 宿泊費記録
- 食事代・雑費管理
- 領収書管理

### その他
- 出張報告書
- 緊急連絡先リスト
- 持ち物チェックリスト
