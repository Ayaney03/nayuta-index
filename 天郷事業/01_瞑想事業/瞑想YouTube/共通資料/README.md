## 🎯 YouTube事業概要

なゆたさんの瞑想指導を動画コンテンツとして配信し、より多くの人に瞑想の価値を届ける事業です。脱スマホ瞑想プログラムとの連携も視野に入れた総合的なYouTube戦略を展開します。
