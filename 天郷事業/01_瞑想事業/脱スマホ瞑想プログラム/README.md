## プログラム概要

脱スマホ瞑想プログラムは、現代人のスマートフォン依存問題を瞑想を通じて解決するプログラムです。なゆたさんの専門知識と実践的なアプローチを組み合わせ、参加者が健康的なデジタルライフを送れるようサポートします。
